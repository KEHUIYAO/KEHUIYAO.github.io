麦迪逊的公园

Posted on 2020-09-22 | Post modified 2021-06-03

Blackhawk Country Club

  • 推荐指数7分

2020/08/31

  • 一个平时无法进入的高尔夫球场，草质很好，平整。
  • 周一关门早，四点就关门了。以后可以周一来，防止被人赶出来。还可以带点饮料零食来这边欣赏日落。

Cross Plains State Park

  • 推荐指数2分

2020/09/16

  • 这公园没路没牌子，野草贼高，底下全是有机肥料？
  • 曾有过隐居山林种地的念头，害，还是太难了。这种狂野让人如何消受？

Devil’s Lake

  • 推荐指数8分

2020/10/10

  • 州立公园，停车费单次$13年卡$28。初次来的时候是社团租了一片场地，组织大家一起破冰，然后爬山结束。当时觉得是个不起眼的小湖。现在经历了很多公园之后再回来看Devil’s Lake，真是不错
  • 北边停车位少，南边多。但是北边可以租船和PaddleBoard。感觉恶魔湖的特点是很多大石块。建议大家的旅行攻略是停在南边，从East Bluff上山一直往北走。在上图中画出了一个红线的捷径，捷径的入口很好找，出口却不是这样。在谷歌地图上看的话，大概是铁轨部分由细转粗的交接部分。途中有一个梯度很大的坡，其实不下也可，整个路程就是一直往西北走，就会渐渐发现一条路。这样可以merge到铁轨人群，单程路线就可以体会到爬山，探险，铁轨，湖边的四重景色。最后在北边租一个PaddleBoard玩一小时回来即可。
  • West Bluff还未探索，我们沿着岸边走的，感觉一般般。如果是南边出发环湖一周+水上运动，会有点累。所以如果是两辆车，建议南北各停一辆，大家从南出发，到北边结束是最好的。

Governor Nelson State Park

  • 推荐指数6分

2020/09/26

  • 州立公园，停车费单次$8年卡$28。开始以为只有一个trail是值得一走的，JZ建议我们去湖边看看。后来发现湖边才是宝藏。适逢风大，有老哥在水上翱翔滑水，很有水平。
  • 草地Trail乏善可陈，湖边有下船的地方，有一片小沙滩可以打排球，还有儿童嬉戏区。不过玩的不多，树林小路也很短。但是鉴于有树有水，勉强及格。

Howard Temin Lakeshore Path

就是指从Nat到Coli的一段沿湖小路，没想到居然还有名字

  • 推荐指数6分

2021/01/16

  • 阳性之后的第一次公园，有意的保持了一点距离。新冠时期难得的出行，路上泥雪交加。老姚想在冰面舞蹈，最后因为岸边冰化得厉害而惨淡收尾。这次有老哥在湖中心站着，不知道在做些什么，看起来湖中心还是很扎实的。希望下次去的时候岸边的冰面也能够结实一点。

Lakeview Park

  • 推荐指数5分

2020/08/23

  • 公园不太大，特色应该是一个喷泉小广场，可惜现在COVID人不多，也不喷了…
  • 中间是一滩死水，不过虫子不太多，旁边还有棒球场和网球场。

Merrill Springs Park

  • 推荐指数6分

2020/08/31

  • 公园很小，不过是去Blackhawk Country Club的必经之路
  • 有一个小井，附近都是豪宅，没什么人

Olbrich Park

  • 推荐指数6.5分

2020/08/29

  • 旁边有个植物园，还可以…如果你的同行者中有植物爱好者的话。
  • 去这个Park主要是为了租船，可以在线预定，整个租赁过程也很舒心。
  • 车有点难停，可能人过多。划船有些累，风很大，水不干净。没有那些大佬级别的景区好。

Owen Conservation Park

  • 推荐指数6分

2020/08/20

  • 距离家很近，里面的路牌不是很清楚，最后绕迷路了…下次应该带着手机
  • 中间有很长的小路，路旁有倒下的大树，很自然。
  • 不过路很难走，基本上90%的时候都在盯着地面看…周围的树木过高，挡住了阳光。属于脚踏实地而不是仰望星空的公园。

Picnic Point

大家对这个地方都很熟了，不过多介绍。

2021/02/03

  • 因为要人少 + 散步 + 学校附近，于是我们选择了此地。Mendota整体结冰，很有感觉，我们就讨论是否可以在冰面徒步。本想从Picnic Point走到码头，可惜只出发了一点的时候，脚下冰面慢慢变得不太扎实，心里小慌，于是打起了退堂鼓。害，英雄气短。收益不多，代价过大。

Raymer’s Cove

  • 推荐指数8分

2020/10/17

  • 探险从一个圆圈型的停车场开始。左手边有一个不知通向何方的小木道。右手边一直通往Picnic Point。秋天时候走在路上，会路过一片金黄色的草地，这感觉像是在梦里。若是你还没有情人，你便会爱上那个迎面走来的女子。

2020/12/12

  • 今天下雪了，在215窝着看《棋魂》好不过瘾。在老姚的坚持下我们还是出门了。雪后的Raymer别有风味，但我们认为不如平时绿色的时候好看。有趣的是树干上的雪被风刮的造型，像剪了一个火箭头发型。

双子湖

  • 双子湖包括两个湖：
    • Tiedman Pond Nature Preserve
    • Stricker’s Pond
  • 推荐指数7.4分

2020/08/23

  • 中间很大的湖，还和旁边的小湖构成了双子湖。因为湖很大，夕阳的时候湖景我感觉还不错。
  • 虽然也是一潭死水，不过味道不太大。湖只要够大，可能味道就不大？
  • 虫子过多！我甚至能听到他们飞行的声音，周围的空气都被震荡起来，很恐怖。我肯定吃进去了几个…
  • 有一个地方很泥泞，把鞋子搞脏了…
  • 整个环岛大约一英里，是个跑步的好地方。

2020/11/07

  • 这次我们还去了双子湖的弟湖。trail设计的很自然，有一段还可以看到一点白白的沙滩，像是Raymer’s Cove的微缩版。
  • 偶遇一个阿姨带着充气小皮艇湖心独自划船，看起来有点不羁。

Tree Farm

  • 推荐指数7分

2020/09/19

  • 这是一个可以自己摘菜的农场，来的季节不同，会有不同的作物。这次来的时候番茄系列还不错。价格很实惠，感觉是超市价格的30%。开过来大概需要单程20min，不过得赶着一波新作物熟了再来比较好。下次或许可以先在网上查查。

Treinen Farm

  • 推荐指数7.5分

2020/10/24

  • 我们在秋天的末尾来体验了一下玉米迷宫，很遗憾玉米虽然成熟了，身旁迷宫却不再青葱。我想夏天来的话一定会更好看，很遗憾只找到了两个punch，下次来要和那一对couple一样，带个笔本，做好笔记，换一张证书！
  • 附近还有trail，不过由于好累就没去（前一天4点才睡…）
  • 有一些花里胡哨的小玩意，塑料套环，鸭子游泳，沙里淘石，喂养牲畜。小孩子来感觉可以玩一天。

Vilas Park

  • 推荐指数7分
  • 体验还行，活动不算多，就是狂走。和学弟摇奖一起去的，疫情初期，动物园还没开。公园里有死水小湖，大片草地，秋千。

2021/01/24

  • 双阴之后第一次坐车出行，公园里人很多，大家在结冰了的湖面上面滑冰⛸，打冰面曲棍球。隔壁湖还有老哥在滑雪。感觉到麦迪逊的冬天对一部分人来说也是很有吸引力的。

野园

  • 推荐指数7.2分
  • Pheasant Branch Conservancy

2020/08/17

  • 从中间的小路开始，，走了北部环线。刚下车迎面而来的就是屎的味道… 夏天的时候温泉池冒泡泡但是没太大意思。期间我们走了两条很窄的小路，地上还有一只死鸽子…小虫乱飞。用老姚的话说，这公园太野了。爬到山顶，豁然开朗，微风习习，算是今日的亮点
  • 我感觉或许这才是自然的样子吧…有些公园环境过好，很舒适，不过有些不太真实。我喜欢说自己很热爱亲近自然，或许有些伪善在里面。这个公园就很「自然」，不过当我真正「亲近」的时候却又收回了手。希望会像爱情一样，终有一天，我会爱上她本来的模样。

2020/11/08

  • 11月初，小路两旁失去了绿色，景色打了折扣。中间的小湖不太流通，看起来脏脏的。
  • 那个温泉还在，还在小喷。那里感觉有点像是活水源头，很清澈。
  • 老姚：「野园还是大」* n
  • 这次我们选择了环园一周，好一通走。我认为还是北部环线好一点。南部环线风景好像不太多。

麦屯留学日记

Posted on 2020-09-22 | Post modified 2021-06-03

2021-06-03

老同桌回国以后，就接到了很多相亲的安排，他为此列了一个名单：复旦一号，西交利物浦一号，交大一号等等。我说：你在排序的时候，其实别人或许也在排序，你可能是交大八号。他深以为然，并掏出了一套理论：一件事，如果你做过，那么也会有很多人做过；如果你只是动过念头，那么也会有很多人做过；如果你根本不想做这件事，还是会有很多人做过。

膝盖还是有点酸痛，但我已经不想再等了，果断约了老哥晚上去打球。老哥一听我又想去打，明则劝我养伤，实则自己想偷懒，但架不住我软磨硬泡，还是得打。记得19年的时候，我的手腕莫名其妙得重伤，当时几乎让我的羽毛球实力腰斩，理论上应该在家多休息来修养，可是我耐不住寂寞，一直频繁出去打篮球，滑雪，导致伤好的特别慢。后来不知道怎么就神奇地恢复了。希望这次膝盖伤势也能神奇的恢复。

2021-06-02

早上一起床立刻查看邮件，等来的是老板的好消息：

大致意思是：“不用管letter上讲的是啥，一切是我说了算。”。哎，终于吃下了一颗定心丸。今天早上看了半场篮网的球，不意外地，篮网淘汰了苟延残喘的凯尔特人，以4-1晋级下一轮，面对的是我的家乡球队密尔沃基buck。期待下一轮阿杜能稍微出点汗，面对同样天赋卓绝的希腊怪兽打出点代表作来。下午依旧很困，狂看摩登家庭死撑，艰难度过，导致的后果就是吃完饭以后更困了，真应该下午睡一觉的。一个人在家感觉效率确实低的可怕，想想可能还不如搬个电脑去KFC学效果好，或许这就是人民的力量，看到人或许就不会那么困？

饭后又去印象城玩了一把投篮机，继续刷新个人最好记录。感慨一项运动从入门到中期往往是最快乐的，原因可能就是前期往往进步最快，获得的成就感也最多，到后期以后，如果没有突破往往就会停滞不前，继而气馁。不知道我的投篮机生涯最终会收敛到一个什么样的水平。

2021-06-01

今天是六一儿童节，百无聊赖的我准备给六月起个好头，开始research的一个月。有趣的是，刚翻开paper准备看，脚就踢了桌子一下，结果导致放在快递盒子上的电子书屏幕轰然坠落，线缆断成两截，屏幕也永远定格在了一页paper上。

我试着修了修，自然是没有修好。只能无奈地从官网上订购了一根新的线缆，不知道何时能够寄到。巧合的是，我想起在去九寨沟的路上，也是正当我拿出电子书准备看paper的时候，安全带的铁扣子给了我电子书屏幕的玻璃板致命的一击，直接导致屏幕碎裂，返厂两个星期，花去700余元。

这两件事仿佛预示着什么，但我决定不信邪，继续research，毕竟我不信玄学那一说。

在车上，广播里突然放出了关于此7月份汪峰苏州演唱会的消息，我关注老汪很久了，这场演唱会我也一直想去听，于是我就和妈妈提了一下。让我意外的事，妈妈居然也想去听汪峰，还告诉我她曾经听过两次汪峰的现场，一次是什么湖南卫视的农民博览会上，还有一次甚至在汪峰还没有那么红的时候，虽然算不上是汪峰的老歌迷，但也算有点熟。我立刻买了两张’再见青春’级别的外场票，期待到时候老汪的表演。

已是深夜了，却忽然发现我的RA appointment只cover了春季学并没有cover暑假。心里突然有一种不好的感觉，急忙发邮件问老板，然后一边写着日记一边焦急等待老板的回复，万一暑假没有funding可就不太妙了，祈祷。

2021-05-31

最近膝盖不好，晚上的羽毛球也全部取消了。吃完后以后骑车去印象城又玩了几把投篮机。这次比上次又进步了一些，把最高纪录刷到了440分，虽然·离大佬四关700+的神级操作还是相差甚远，但我慢慢在摸索中掌握这项运动的诀窍，也感觉很有意思。投了五六把以后手就酸的不行，感觉这投篮机似乎比举铁更练习肱二头肌，遂结束了今天的投篮机训练。走出印象城，天还没黑，今天的晚霞挺美，随手拍了一张照片。

随后骑车去附近的85度C面包店买了三个辣味肉松面包，花了23元。想到之前在麦迪逊面包店买的肉松面包，一个就要3刀多，真是贵。一路骑车看着常熟的夜景，想起老王上次来常熟，要不是天太晚加上前一天没休息好，真应该带他在常熟市里骑一圈的，也不至于老王在以后提起常熟的时候只能记得那条三环高架路了。

希望膝盖能快点好起来，没有球打的日子有些无聊。

2021-05-30

奶奶烧饭总是很快，总是能在半小时内烧出四到五个菜。本来从小到大我都觉得这是一件很自然的事情，直到我出国独立做饭之后，才发现这是一件多么困难的事情。拿我自己来说，除去买菜，看视频学烧菜的时间，真正上手做两道菜最少也需要40分钟。惊讶之余，我多次进厨房看奶奶操作，发现奶奶做菜总是有条不紊，条理特别清晰，炒锅炖锅并排操作，所有的原材料好像都在手边，无需寻找，加上几十年的熟练度，一道菜接着一道菜，像是没有间隔的做出来。想起奶奶曾提起过她年轻的时候经常烧十几个人一起吃，我脑中不由浮现出在国外的时候，一位学弟招待我们六个人吃饭，从下午4点烧到8点的壮烈场面。奶奶这可是给十几个人烧饭啊，真是难以想象。哎，老一辈的勤劳与智慧真是有太多可以学习的东西。

说起烧新菜，奶奶也有一套。她不上网，看不了美食作家王刚，可她们有很老派的学习方法：口口相传。那天奶奶说学了个新菜，奥尔良烤鸡翅。我将信将疑，感觉奶奶把奥尔良用常熟话念出来就已经令我称奇了。鸡翅自然是很美味，虽然味道不那么奥尔良。奶奶说这是她和邻居黄阿姨学的，黄阿姨做给自己家里的小孙子吃，小孙子很爱吃，于是奶奶就问她学了一下，然后烧给我吃。至于黄阿姨是哪里学来的，我也就不得而知了。

Leetcode stack problem

Posted on 2019-03-26 | Post modified 2020-07-23

单调栈

Find the first element on the left which is larger than itself.

The most straight forward idea is to compare the current element with the left element in the list one by one. But soon we found that it is not the efficient solution. For example, you have a list [1,4,2,3,5]. When you compare the element 5 with 3, you found 5 > 3, then you proceed to the next element 2. But when you set the position of 3, you have already do the comparison 3 > 2. So we do not need to do the comparison 5 > 2, which could save the computation time if the list is much larger.

So, we apply the trick of the decreasing stack. The idea is to keep a temporary list and the elements in this list are in decreasing order. At the beginning, the decreasing stack is empty. Then, we loop through the list, and for each element, we compare the element with the top element in the decreasing stack. If larger, then pop the top element out, and do it again until the decreasing stack is empty or it encouter the top elment which is larger than itself. If empty, it means no other element is larger than it, otherwise, output the element which is larger than it. Finally, push it into the decreasing stack.
