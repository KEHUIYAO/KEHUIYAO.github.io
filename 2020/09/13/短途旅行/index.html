短途旅行

Posted on 2020-09-13 | Post modified 2020-09-14

这次旅行定在了刚开学的第一个周末，本来是可以放在暑假的，但由于我有一个qualify exam卡着，8月份出去估计也没心情尽情地玩，于是干脆就放到考完qualfiy之后。旅行计划都交给了老王，老王素来稳妥，很早就把计划贴到了博客上供我们参阅。准备计划的工作交给了老蒋，老蒋作为安排届的master，给了列了一个长长的清单，我一考完试，就有条不紊地按照清单上的任务一样样执行，到了出发前夕，竟是做完了。

我很喜欢这种被安排好了的旅行，也许和小时候总是和爸爸妈妈出门旅行有关，啥事也不用操心。不过来美国三年也旅行了几次，去佛罗里达那次印象深刻，我和老刘用一个晚上定完了机票酒店，租车和行程，后续还有诸多变数，还是有点小累。去纽约那次找高中同学可谓体验极好，纽约地头蛇给我在他家安排了一张气垫床，一劳永逸地解决了我的住宿问题，于是我在纽约停留了7天之多，每天只看一两个景点，也硬是把纽约逛的太差不差了。

这次旅行的主要目的还是为了亲近自然，目的地大多是state park，手法大多是hiking。鉴于是老王安排的路线，hiking的路线一定是该公园最难的trail，甚至还有可能是在没有路的地方开辟一条新路，这是我有过心里准备的。不过我还是很喜欢挑战这种未知的刺激，只要不是一念天堂的刺激，我还都可以接受。

果然，到了第一个公园,Mauoketa caves state park，老王就掏出了一条完全没有人走的路。从一个被杂草覆盖的入口，我们欣然前往，后面有许多外国老哥看我们走了这条路，也跟了上来，不知道他们后面有没有后悔做这个决定。这条路险峻异常，好多地方都是两边悬崖，中间一块石头，我的鞋有点小滑，只能用手扒着旁边的杂树保持平衡，也算有惊无险。这个公园的特色是洞穴地貌，也因此在爱荷华公园排行榜上占据首席，但我们在查攻略的时候得知由于疫情的原因，这些cave都关了，当时不免有些失望。但到现场一看，不由大喜。所谓关了，其实就是在入口用一根长木条拦住，一跨就进去了。有的洞穴初极狭，才通人，复行数十步，豁然开朗，甚至可以建一个篮球场。还有的洞穴入口就像一个狗洞，没有人敢去尝试，让我不由得想到如果我拥有张无忌的缩骨功或许可以尝试一下，不然怕是爬了进去卡在里面，退不出来。总体来说，由于接近三年没看到这种洞穴地貌（不得不说国内这种风景还是多），还是对这个公园留下了非常不错的印象。又一想，一普通的state park就能有如此小景，我没去过的黄石，大峡谷，这些national park该有多壮观，有机会一定要去一去。

此行的第二个公园并没有给我留下深刻的印象，暂且不提，但第三个公园backbone state park老蒋尤其喜欢，或许是觉得这个公园的trail在像我们madison优美的ramer’s cove? 老王意料之中地掏出了这个公园最难的trail，名曰Six trail, 这条trail的入口和出口之间有一个不小的海拔差。入口在高地，出口在低地。当我们花了接近20分钟从入口走到出口以后，我们发现如果不原地返回，需要绕很大一个圈子才能走回我们停车的地方。于是我们四个男生自告奋勇，决定原路返回去拿车。诚不想海拔差岂是那么容易克服的，往回走的路上简直就是在爬山，累的我们气喘吁吁，大汗淋漓。这个公园最著名的是一个沙滩，但因为有很多鹅在上面拉屎，在最近别人的游记上可谓臭名昭著。我们抱着试一试的心态欣然前往，却发现鹅屎基本都被清理掉了，沙滩上挺空矿的，风景也不错，倒也算是一个小惊喜。

当天下午，我们去尝试了漂流。具体流程是老王联系一个老哥，并和老哥约好我们到达漂流地点的时间，然后老哥用自己的车拉了六条船过来。然后我们每人一条船，从河的高处慢慢往下漂，直到一个标志处停下，全程约三小时左右。我和老刘提前把车开到了河的下游，也就是还船的地方，老哥用他的车把我们拉了回来，一来一回折腾了半个小时之多，途中一度觉得老哥是不是人贩子，会不会突然掏出一把枪，事实证明我们多虑了。

漂流开始了，水很平缓，要获得充足的前进动力还是得靠摇桨。在过一个二岔水道的时候，老唐选择了右边，很不幸的被一股涡流卷住，整条船撞到了树上，人仰船翻。更不幸的是，由于水流冲击力过大，老唐的船卡在了树里，拔不出来了。学弟负责断后，见此情景毫不犹豫的赶去救援，无果。老王，老蒋，老刘和我冲在了最前面，起初根本不知道后面发生了什么，因为迟迟看不到老唐和学弟，也猜到了事情不妙。老刘当机立断，选择了划船折返。由于逆水行舟，老刘拼尽了全力，在五分钟内前进了不到30米，我们从后面看去，一度觉得老刘在原地静止不动了。不过通过不懈的坚持，老刘最终还是赶到了事发现场。但听说一到那他的船就翻了，自己受了小伤，无功而返，不过show了effort，江湖义气到位了。我，老王和老蒋醒悟过来地太晚，已经无力回头救援，只能把船往岸边一靠，上岸安心等待。过不多久受伤的老刘先行折返，和我们说明了情况。又过了20分钟左右，老唐和学弟姗姗来迟。听老唐说，是一帮外国老哥把她的船解救了出来。外国老哥也是全力以赴，在救船的过程中甚至损失了一部手机，真是令人感慨世间的善意。小插曲过后是一路坦途，我们泛舟湖上，体会着自然的美好，可谓轻松惬意。

漂流结束后，我们按照原计划去往了此行的第四个，也是最后一个公园Pikes peak state park观赏日落。这个公园同样没有给我留下很深的印象，但我隐约觉得有了一种在家乡常熟爬虞山的感觉。很遗憾的是，由于被两旁树木遮挡，我们全程都没有看到日落。但我们却意外地看到了一汪小泉水。老刘美其名曰黄果树瀑布，可谓山中无老虎，猴子称大王。
