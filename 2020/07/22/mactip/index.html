Mac优秀软件以及快捷键分享

Posted on 2020-07-22 | Post modified 2020-07-24

软件类

上淘宝购买叶子登录器：https://item.taobao.com/item.htm?spm=a1z09.2.0.0.344d2e8dYhbcqt&id=586441192815&_u=22c1nhn75862，通过这个登录器可以内购商家下载的所有软件。（如pdf expert等神器）。

如果想花钱支持一些软件可以分开购买。

效率类：

Volta: 可以控制mac cpu功率，使得续航长一倍

paste：剪贴板增强器，必装。

Yoink: 极其方便的软件，提供类似于windows上的control + x，必装。

pdf expert: 太强不提。

goodnotes：有ipad的话联动还不错。

Fantastic2：最好的日历。

Mathpix: 图片转换成latex code，写作业神器。（网上下载，免费）。

go2shell：直接可以在finder里打开terminal，路径就在folder里。

typora：笔记软件，markdown格式写笔记。

blink：护眼软件？

NTFS for Mac： 哪天u盘识别不了你就会用它了

PuyoVs: ??? 别问我为什么会出现

视频：

splayer（装一个就行，比vlc好用）。

代码类:

Sublime: 轻量级代码编辑器，网上下载，免费。

coderunner：付费的，但叶子有。

gemini：代码查重的

Pycharm？

Rstudio？

性能：

volta：？ 限制功率，防止mac风扇乱转。防止chrome狂吃你的内存。

快捷键：

finder 中的tap功能

这样就不用开几个finder了，使用方法是双击打开文件夹之前按住command。

用spotlight搜索文件以及进入文件所在目录

  • To search by file type, use the word “kind” and the file type. For example, type “kind:folders” or “kind:audio”.
  • To show the location of a file on your Mac, choose the file from the results list, then press and hold Command. The file’s location appears at the bottom of the preview. To open the file’s location, press Command-R.

文件移动或是拷贝（借助yolink）

假设你想把一个文件从A移动到B，可以通过yolink直接拖放完成。但A中的文件会消失，加入你只是想拷贝一份，那就在把文件拖入yolink前按住option+command吧！
